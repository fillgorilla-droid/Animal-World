# Animal World

Welcome to Animal World — a simple educational website about animals.

## About the project

Animal World is a basic web project created using only HTML and CSS. The goal of this website is to present information about animals in a clear and simple way.

On this website you can learn:
- facts about different animals
- information about their childhood and development
- details about some of the largest animals in the world
- structured information presented in tables and sections

This project was created for learning purposes to practice HTML and CSS and improve front-end development skills.

## Features

- Information about animals
- Table with facts
- Simple and clean layout
- Structured content sections
- Basic responsive design

## Technologies used

- HTML5
- CSS3

No JavaScript or external libraries were used.
